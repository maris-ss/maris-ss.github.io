Sobre o site

Como editar o dashboard?

Para editar o dashboard, arraste os widgets para o local desejado e fixe-os, se quiser, clicando no ícone de fixar à direita.

Como resolver uma divergência?

Para resolver a divergência, basta acessar as notificações no menu superior, onde será indicado se algo está errado, permitindo que você tome as ações necessárias para corrigir o problema.

Como editar os Widgets?

Para editar os widgets, basta você clicar no sinal de "+" que aparece ao lado deles. Ao fazer isso, as opções de edição estarão disponíveis, permitindo que você personalize os widgets conforme deseja.

Sobre finanças

O que é um PER/COMP?

O PER/COMP é um documento para solicitar a devolução de impostos pagos a mais ou errados, ou para usar créditos tributários em dívidas de impostos federais.

O que são compensações?

É o processo que permite ao contribuinte usar créditos tributários para abater ou quitar dívidas de outros tributos.

O que é e-CAC?

O e-CAC é uma plataforma online da Receita Federal que oferece serviços fiscais, como consulta de situação fiscal, envio de documentos e geração de guias de pagamento.
